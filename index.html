Endpoints

GET /api_status/

Verifica se a API está ativa ou em manutenção.

{ "status": "success", "data": null, "time_response": 1747927944, "api_version": "1.0.0" }

GET /get_all_clients/

Busca todos os clientes no banco de dados.

{ "status": "success", "data": [...], "total_clients": 200, "time_response": 1747928054, "api_version": "1.0.0" }

GET /get_client/

Pega todos os dados do cliente cujo ID é passado como parâmetro.

GET /get_client_by_city/

Pega os clientes cuja cidade é igual ao parâmetro passado.

GET /get_client_by_email_domain/

Pega os clientes cujo domínio do email corresponde ao parâmetro passado.

GET /get_total_males_and_females/

Retorna o total de clientes por gênero.

POST /add_new_client/

Adiciona um novo cliente.

{ "nome": "", "sexo": "", "data_nascimento": "", "email": "", "telefone": "", "morada": "", "cidade": "", "ativo": true }

PUT /update_client_name/

Atualiza o nome do cliente.

{ "nome": "", "id": "" }

PUT /update_client_email/

Atualiza o email do cliente.

{ "email": "", "id": "" }

PATCH /delete_client/

Deleta o cliente pelo ID.
